Technical Support

What should I do if the website is not loading?

If the website is not loading, try these steps:

  • Check your internet connection to ensure it's stable.
  • Clear your browser cache and cookies.
  • Try accessing the website using a different browser or device.
  • Disable any browser extensions that might interfere with website loading.
  • If the issue persists, contact our technical support team.

How do I report a bug?

If you encounter a bug, please report it by following these steps:

  1. Go to the "Help" section on our website.
  2. Select "Report a Bug".
  3. Fill out the form with a detailed description of the issue.
  4. Submit the form, and our technical team will investigate the problem.
